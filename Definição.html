Explicações Tech Simplificadas

Servidor FTP

FTP (File Transfer Protocol) é um protocolo usado para transferir arquivos entre um cliente e um servidor na internet.

  • Usado para publicar sites (enviar arquivos HTML, CSS, JS para um servidor web).
  • Funciona com credenciais (usuário e senha) ou acesso anônimo.
  • Exemplos de clientes FTP: FileZilla, WinSCP, Cyberduck.

Exemplo de conexão FTP: ftp://exemplo.com

JavaScript

JavaScript (JS) é uma linguagem de programação usada para criar interatividade em páginas web.

  • Roda no navegador do usuário (front-end).
  • Pode manipular HTML/CSS, validar formulários, criar animações e muito mais.
  • Também usado no back-end com Node.js.

Exemplo de código JS:


function saudacao() {
    alert("Olá, Marinke!");
}

Docker

Docker é uma plataforma que permite criar, executar e gerenciar aplicações em containers.

  • Containers são ambientes isolados e leves que empacotam uma aplicação e suas dependências.
  • Facilita a implantação e evita problemas do tipo "funciona na minha máquina".
  • Usa imagens Docker (ex: Ubuntu, Node.js, MySQL) para criar containers.

Exemplo de comando Docker: docker run -d nginx
